尾递归：防止内存爆炸

  • 只返回函数本身。
  • 量放在参数中。
    Ex:
fun func(_ n :Int, _ total :Int = 0) -> Int{
    if (n == 1)
        return total + 1
    else
        return func (n - 1, total + n)
}
