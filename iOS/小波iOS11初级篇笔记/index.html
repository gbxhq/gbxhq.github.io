新场景

  1. 拖一个新的UITableViewController，就要新建对应的场景控制器。
  2. New File –> Cocoa Class –> 起个名,父类选择UITableViewController
  3. 在StoryBoard中，identity栏讲场景与类绑定

添加导航：

菜单栏-Editor–>Embed in–>Nav~
