#68.验证回文字符串 Ⅱ

排除一个字符只让一个标记移动的时候。要判断清除。(不好形容)

比如字符串cuucu。s[0]≠s[4]…

但是s[1]=s[4]..这时候如果让第一个标记右移，新字符串是uucu就错了。

因为其实s[0]=s[3]..应该让第二个标记左移。得到新串cuuc。
