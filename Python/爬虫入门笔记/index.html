Problems

TypeError: expected string or bytes-like object

对着网上教程，开门就遇到这个报错。
搜了下，原来是
re.findall(r"<title>(.+?)</title>", html)
这种写法是Python2的。
在Python3中只需要改写成
res = re.findall(r"<title>(.+?)</title>", str(html))
